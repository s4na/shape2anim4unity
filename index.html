About Unity Animation Editor

What is this tool?

Unity Animation Editor is a web-based tool for Unity developers to edit and manipulate Unity animation files (.anim) directly in the browser. No installation required, everything runs locally in your browser.

Features

  • Text to File Converter: Paste Unity animation YAML content and download it as a .anim file
  • Remove Zeros: Upload an animation file and remove zero-value keyframes to optimize file size
  • BlendShape Import: Convert Unity BlendShapeWeights JSON data to animation files
  • Privacy-Focused: All processing happens in your browser - no data is sent to any server
  • Open Source: Free to use and modify under MIT License

How to Use

Text to File Converter

  1. Copy the YAML content from your Unity animation file
  2. Paste it into the text editor
  3. The tool will automatically validate the format
  4. Enter a filename (optional)
  5. Click "Download .anim" to save the file

Remove Zeros Tool

  1. Upload or drag-drop your .anim file
  2. Configure processing options (threshold, curve types, etc.)
  3. Click "Process Animation"
  4. Review the statistics showing what was removed
  5. Download the optimized file

BlendShape Import

  1. Copy BlendShapeWeights JSON data from Unity
  2. Paste it into the text editor
  3. Configure animation options (name, mesh path, duration)
  4. Click "Generate Animation"
  5. Download the .anim file

Browser Compatibility

This tool works best on modern browsers:

  • Chrome/Edge 90+
  • Firefox 88+
  • Safari 14+
