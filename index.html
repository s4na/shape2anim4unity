Unity アニメーションエディターについて

このツールについて

Unity アニメーションエディターは、Unity開発者がブラウザで直接Unityアニメーションファイル（.anim）を編集・操作するためのWebベースのツールです。インストール不要で、すべてブラウザ内でローカルに動作します。

機能

  • テキストからファイルへのコンバーター: UnityアニメーションのYAML内容を貼り付けて、.animファイルとしてダウンロード
  • ゼロ値の削除: アニメーションファイルをアップロードして、ゼロ値のキーフレームを削除し、ファイルサイズを最適化
  • BlendShapeインポート: Unity BlendShapeWeightsのJSONデータをアニメーションファイルに変換
  • プライバシー重視: すべての処理はブラウザ内で行われ、データはサーバーに送信されません
  • オープンソース: MITライセンスのもと、無料で使用・改変可能

使い方

テキストからファイルへのコンバーター

  1. UnityアニメーションファイルからYAML内容をコピー
  2. テキストエディターに貼り付け
  3. ツールが自動的にフォーマットを検証
  4. ファイル名を入力（任意）
  5. 「.animをダウンロード」をクリックしてファイルを保存

ゼロ値の削除ツール

  1. .animファイルをアップロードまたはドラッグ＆ドロップ
  2. 処理オプション（しきい値、カーブタイプなど）を設定
  3. 「アニメーションを処理」をクリック
  4. 削除内容を示す統計を確認
  5. 最適化されたファイルをダウンロード

BlendShapeインポート

  1. UnityからBlendShapeWeightsのJSONデータをコピー
  2. テキストエディターに貼り付け
  3. アニメーションオプション（名前、メッシュパス、継続時間）を設定
  4. 「アニメーションを生成」をクリック
  5. .animファイルをダウンロード
